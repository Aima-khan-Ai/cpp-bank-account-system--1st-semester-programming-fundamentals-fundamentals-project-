# # Bank Account System (C++)

This is my 1st Semester Project for Programming Fundamentals subject made in C++.
This project was made by Aima.

## About
This is a simple console-based bank account system built in C++. The user can set an opening balance at the start and then perform basic banking operations like depositing money, withdrawing money, and checking the current balance. The program keeps running until the user chooses to exit. This project was made to practice the basic concepts of C++ programming that we studied in our 1st semester.

## Features
- Set opening balance at the start
- Check current balance anytime
- Deposit any amount into the account
- Withdraw money from the account
- If the user tries to withdraw more than the balance, the program shows an insufficient balance error
- If the user enters an invalid amount (like 0 or negative), the program shows an invalid amount error
- The menu keeps repeating until the user presses 4 to exit
- Simple and easy to understand interface for beginners

## Concepts Used
- Variables and Data Types (double, int)
- If/Else If/Else Statements
- Do/While Loop
- Basic Input and Output (cin and cout)
- Conditional Logic and Input Validation

## How to Compile and Run

Step 1 - Open terminal in the project folder

Step 2 - Compile the code:
g++ bank_account.cpp -o bank

Step 3 - Run the program:
./bank

Step 4 - Enter your opening balance and use the menu to perform operations

## How to Use
1. When the program starts, enter your opening balance
2. A menu will appear with 4 options
3. Press 1 to check your current balance
4. Press 2 to deposit money into your account
5. Press 3 to withdraw money from your account
6. Press 4 to exit the program
7. After every deposit or withdrawal, the updated balance is shown on screen
8. If you enter a wrong choice, the program will ask you to try again
